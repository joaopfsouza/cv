# Currículo

## Install RenderCV

```bash
uv tool install "rendercv[full]"
```
## Create CV Yaml

```bash
rendercv new "cv_joao_paulo"
```

## Render CV

```bash
rendercv render "cv_joao_paulo.yaml"
```

render in real time
```bash
rendercv render --watch "cv_joao_paulo.yaml"
```
